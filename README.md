# DNA Motif Analyzer

Program analizuje sekwencję DNA i wyszukuje określony motyw.

## Funkcje
- wczytywanie FASTA lub TXT
- wyszukiwanie motywu
- liczba wystąpień
- wizualizacja pozycji
- eksport wyników do CSV
- GUI w tkinter

## Instalacja

pip install -r requirements.txt

## Uruchomienie

python main.py
